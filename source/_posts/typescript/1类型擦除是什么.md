为啥需要类型擦除？

因为主流的执行器都是只支持 JavaScript，所以需要将 TypeScript 转为 JavaScript。



将ts 转成 js 的过程叫做**类型擦除**。



哪些方式做到

| 类型擦除类型 |          | 编译速度 |
| ------------ | -------- | -------- |
| esbuild      | 只是清除 | 快       |
| swc          | 只是清除 | 快       |
| tsc          |          | 慢       |
| babel        |          | 慢       |
